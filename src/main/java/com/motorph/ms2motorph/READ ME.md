Program Details
The MotorPH Payroll System is a Java-based program designed to compute the payroll of employees using employee information and attendance records stored in CSV files.

The system performs the following tasks:
Reads employee data from the MotorPH Employee Data CSV file
Reads attendance records from the Attendance Record CSV file
Computes the total hours worked by each employee
Calculates the employee's gross salary
Applies necessary deductions
Computes the final net salary

This system integrates all modues developed from Week 5 to Week 8 of the course project.
